## Playing the game

The sample game can be played at: [gameatron.sandermolenkamp.com](https://gameatron.sandermolenkamp.com)

### Keyboard shortcuts:

- `q`: quite game 
- `.` or `space`: skip dialogue line 
- `s`: save game
- `l`: load saved game
- `r`: refresh game script
